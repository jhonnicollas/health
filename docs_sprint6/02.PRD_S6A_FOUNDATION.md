## 7.1 AI Operating Mode Config (Super Admin)

```text
2 new system configs seeded for operating mode control:

clinicalCopilot.operatingMode = 'standard' (default)
  Values: 'standard' | 'proactive' | 'super_aktif'
  Controlled by: Super Admin only
  Change requires: admin.aiConfig.update permission + medical reviewer approval
  
clinicalCopilot.operatingModeChangeRequiresMedicalReviewer = 'true' (default)
  Values: 'true' | 'false'
  When true: mode change must be approved by medical reviewer before生效
  Audit: HL_auditLogs action='aiOperatingModeChanged'
```

Per §0.3 of main PRD:
- `standard`: default, safest mode. No diagnosis final, no resep, no dosis, no specialist claim.
- `proactive`: AI boleh diagnosis final. No resep, no dosis, no specialist claim.
- `super_aktif`: AI boleh diagnosis final, resep, dosis, specialist claim. Medication change always blocked.
