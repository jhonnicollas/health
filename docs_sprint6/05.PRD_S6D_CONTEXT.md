## 4. Data Sufficiency Score

```text
Score = weighted sum of available data:
- Profile complete (sex, heightCm, age) : 10 points
- Measurements last 7 days              : 25 points
- Measurements last 30 days             : 15 points
- Symptoms logged                       : 15 points
- Medication records                    : 10 points
- Vectorize memory present              : 10 points
- Hydration data                        : 5 points
- Cycle data (if applicable)            : 5 points
- Safety events history                 : 5 points
Total max: 100

Score 0-30  → data sangat terbatas
Score 31-60 → data terbatas
Score 61-100 → data cukup
```

## 7. Acceptance Criteria

```text
1. Full §9.3 context package built from D1 + Vectorize + AI Search.

3. Data sufficiency score calculated correctly (0-100).
4. Consent filter gates hydration + cycle fields.
5. Context trace records all data sources used.
6. forbiddenActions list populated based on disclaimer acknowledgment.
```
